LocalStorage JS

A powerful localStorage utility by Ravi Pandey

Why Use This Package?

  • Simple API for `setItem`, `getItem`, `clear`, and `backup`
  • Set expiry on keys and auto-remove after time
  • Backup & Restore localStorage to/from JSON
  • Debug versions of set/get with logs
  • Works with React, Vue, Angular, Vanilla JS

Live Demo (Check Console)

Install Package from NPM

                    npm install @mrravipandee/localstorage-js

Use this sample code:

                    import { storage } from 'localstorage-js';

                    storage.setItem('user', { name: 'Ravi' });
                    console.log(storage.getItem('user'));

                    storage.setWithExpiry('token', 'abc123', 5);
                    console.log(storage.getWithExpiry('token'));

                    const backup = storage.backup();
                    storage.clear();
                    storage.restore(backup);

                    storage.debugSet('email', 'imravipanday@gmail.com');
                    storage.debugGet('email');
